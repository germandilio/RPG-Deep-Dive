    Namespace Utils.UI.Dragging

    Classes

    DragItem<T>

    Allows a UI element to be dragged and dropped from and to a container. Also support auto swap on mouse right click.

    After the item is dropped it will be automatically return to the original UI parent. It is the job of components implementing IDragContainer, IDragDestination andIDragSource` to update the interface after a drag has occurred.

    Interfaces

    IDragContainer<T>

    Acts both as a source and destination for dragging. If we are dragging between two containers then it is possible to swap items.

    IDragDestination<T>

    Components that implement this interfaces can act as the destination for dragging a DragItem.

    IDragSource<T>

    Components that implement this interfaces can act as the source for dragging a DragItem.
